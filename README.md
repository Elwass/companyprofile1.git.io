# Aurora Dine & Brew – Company Profile

Website statis bertema food & beverage dengan kesan elegan yang menampilkan profil perusahaan, signature menu, layanan, testimoni, galeri, serta informasi kontak.

## Struktur
- `index.html` – halaman utama.
- `assets/css/style.css` – gaya visual bernuansa biru.

## Penggunaan
Buka `index.html` pada peramban modern untuk melihat tampilan website.
